## 转义字符
```
print('hello\nworld')    #\n--->换行
print('hello\tworld')     #\t--->空出一个制表位，一个制表位是4个字符
print('hello\bworld')    #\b--->向前覆盖一个字符
print('hello\rworld')     #\r--->向前覆盖到行首
print(r'hello\nworld')   #r或R--->表达式不表达，原样输出
```
