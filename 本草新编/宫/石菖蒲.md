# 性味归经
- 味辛而苦，气温， 无毒
- 心经
- 石上生者良，九节更佳（取九节通九窍之意）
# 功效
- **开心窍**
- 善通气，止遗尿，安胎除烦闷，治善忘
# 用法
## 开心窍
- [[人参]]为君补虚，石菖蒲为佐开心窍
## 治健忘
- 用法同上
