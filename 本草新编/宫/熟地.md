# 性味归经
- 味甘，性温，沉也，阴中之阳，无毒
- 肝、肾经
# 功效
- 生血益精，长骨中、脑中之髓
- 生真阴之气，降虚火之焰
- 逐心、肝、肾之痰，助脾、肺之痰
# 用法
## 配伍
- 阳药可以奇用，阴药必须耦用
- | 目的         | 配伍用药          |
| -------------- | -------------- |
| 补血 | 当归、白芍、川芎 |
| 补  | 消骨中淤血 |
| 大便燥结 | 下硬粪 |
| 产后亏损 | 除血晕 |
| 肝中血燥 | 解纷 |
| 心中血枯 | 润泽 |
| 脾中血干 | 滋养 |
| 心中血枯 | 润泽 |
