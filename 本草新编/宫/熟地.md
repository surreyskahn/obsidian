# 性味归经
- 味甘，性温，沉也，阴中之阳，无毒
- 肝、肾经
# 功效
- 生血益精，长骨中、脑中之髓
- 生真阴之气，降虚火之焰
- 逐心、肝、肾之痰，助脾、肺之痰
- 生新血
# 用法
## 逐心、肝、肾之痰，助脾、肺之痰
- 熟地入胃后，胃见其为肾经之药，便引入肾经，不会腻膈
- 心火郁熟地可以逐心、肝、肾之痰
## 配伍
- 阳药可以奇用，阴药必须耦用
- | 目的         | 配伍用药          |
| -------------- | -------------- |
| 补血 | 当归、白芍、川芎 |
| 补心肾，使水火既济 | 人参 |
| 补脾肾之亏 | 白术 |
| 滋将枯之肺肾 | 麦冬、五味子 |
| 益将绝之肝肾 | 白芍 |
| 补火衰之命门 | 肉桂 |
| 平膻中之沸火 | 枣仁 |
| 清大肠之血 | 地榆 |
| 凉胃中之火 | 沙参 |
| 泻阳明之火 | 元参 |
| 产后血晕诸病 | 人参、当归 |
- 以上表中药物使用时，熟地用一两~二两为君，配伍之药五钱~八钱，其中肉桂只可以用一钱~二钱，最高不超过三钱
