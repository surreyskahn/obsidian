# 性味归经
- 味甘，性温，沉也，阴中之阳，无毒
- 肝、肾经
# 功效
- 生血益精，长骨中、脑中之髓
- 生真阴之气，降虚火之焰
- 逐心、肝、肾之痰，助脾、肺之痰
