# 性味归经
- 味甘，气微寒，降也， #阳中微阴 ，无毒
- 手太阴肺经、手少阴心经
# 功效
- 泻肺中之伏火，清胃中之热邪，补心气之劳伤，止血家之呕吐
- 益精强阴，解烦止渴，美颜色，悦肌肤
- 退虚热神效，解肺燥，定喘咳
# 用法
## 必须大量使用
## 安补肺气
- 肺之气，夜必归于肾；肾之气，昼必归于肺
- 麦冬能安肺气，使得肺气归于肾中，但不能补肾。但金生水，肾脏还要取济于肺脏，肺仍不安。所以还要配伍补肾之药——**[[五味子]]**，取其补肾之意，并非敛肺。
- 补肺气时注意胃气，配伍温胃之药
## 救肺气
- 脾胃已绝，肺不能生；肾经已绝，肺无法养肾，无药可救
- 胃气不绝 有可救之机
>麦冬为君，加入[[人参]]，[[熟地]]，[[山药]]，[[山茱萸]]
