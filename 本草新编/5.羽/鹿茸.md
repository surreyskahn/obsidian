# 鹿茸
- 味甘、咸、苦、辛，气温，无毒
- 益气滋阴，扶肢体羸弱，强志坚齿，止腰膝疼痛，破流血隐隐作痛，逐虚劳
- 治女人崩漏下血
- 疗小儿寒热惊痫，塞尿血泄精，散石淋痈肿
-==阳痿不坚==
# 鹿角
- 味淡，气温
- 逐鬼辟邪，轻身益气，续绝伤，强筋骨，消痈疽，愈恶疮
- 止妇人梦与鬼交
# 鹿角霜
- 专止滑泄
# 鹿肾
- 补中以滋元
# 鹿血
- 调血脉，止腰疼
- 滚酒调热服用，禁止生用
