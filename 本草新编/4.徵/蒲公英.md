# 性味归经
- 味苦，气平，无毒
-  #胃经 、 #脾经 
# 功效
- 溃坚肿，消结核，解食毒，散滞气
- 泻胃火
# 用法
## 泻胃火
- 白虎汤泻胃火恐伤胃气，用蒲公英能泻火而不伤胃，多用至一两少用至五钱
## 蒲公英与金银花
|      | 蒲公英         | [[金银花]]   |
| ---- | -------------- | -------- |
| 归经 | 阳明经、太阴经 | 无经不入 |
|      | 攻多于补       | 补多于攻 |
- 合用功效更佳
## 恶刺伤或狐尿刺
- 蒲公英根茎白汁，涂疮口上，根叶一两煎汤，内外合治
## 蒲公英膏
- 百斤蒲公英，可以取膏7斤 P186
