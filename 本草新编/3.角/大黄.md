# 性味归经
- 味苦，气大寒， #阴中之阴 ， #降 ，无毒
- #胃经 、 #大肠经 ，配伍佐使药可达各经
# 功效
- 荡涤积滞，调中化食，通利水谷，推陈致新
    - 导淤血，滚痰涎，破癥结，散坚聚，止疼痛
- 败痈疽热毒，消肿胀
# 用法
- 性甚速，走而不守
## 炮制
| 走向 | 炮制方法 |
| ---- | -------- |
| 上升 | 酒制     |
| 下行 | 加入芒硝 |
| 速驰 | 生用     |
| 平调 | 熟煎     |
| 少留 | 加入甘草 |
- 承气汤中加入[[人参]]、[[当归]]，补气之药不可以大量加入，补血之药要大量加入
>大黄荡涤肠胃要有水才能载舟下行，无水要补气滋阴养血
