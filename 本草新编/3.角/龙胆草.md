# 性味归经
- 味苦涩，气大寒， #阴 ，无毒
# 功效
- 专于利水消湿，除湿热黄疸
- 治目，止痢，退热，却肿
# 用法
- 过于分利，恐耗气血
## 湿热、黄疸类比
| 湿热                           | 黄疸                             |
| ------------------------------ | -------------------------------- |
| 热结膀胱                       | 火炎于肾脏                       |
| 急用苦寒之龙胆草，以泻膀胱之热 | 急用[[黄柏]]、[[知母]]以泻热 #？ | 
- 湿热入肝易散，入于胆湿难祛
- 入于胆配伍[[柴胡]]，舒胆中之气，使湿热之邪从外渗出
